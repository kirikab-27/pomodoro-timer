# ポモドーロタイマー

集中力を高めるポモドーロテクニック用タイマーアプリケーション。

## 機能

- 🍅 **ポモドーロタイマー**: 25分の作業セッションと5分/15分の休憩
- ⏱️ **カスタマイズ可能**: タイマーの長さを自由に設定
- 📊 **統計機能**: 作業履歴と生産性の可視化
- 🌙 **ダークモード**: 目に優しいダークテーマ対応
- 🔔 **通知機能**: セッション完了時の音声通知
- 📱 **レスポンシブデザイン**: PC・タブレット・スマホ対応

## 技術スタック

- **フロントエンド**: Next.js 15 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **データベース**: Prisma + SQLite
- **状態管理**: Zustand
- **UIコンポーネント**: Lucide Icons

## セットアップ

### 必要要件

- Node.js 18.0.0以上
- npm または yarn

### インストール手順

1. リポジトリをクローン:
```bash
git clone <repository-url>
cd pomodoro-timer
```

2. 依存関係をインストール:
```bash
npm install
```

3. データベースをセットアップ:
```bash
npm run db:generate
npm run db:push
```

4. 開発サーバーを起動:
```bash
npm run dev
```

5. ブラウザで開く:
```
http://localhost:3000
```

## 使い方

1. **作業開始**: 「開始」ボタンをクリックして25分のポモドーロセッションを開始
2. **休憩**: 作業セッション完了後、自動的に休憩タイマーに切り替わります
3. **カスタマイズ**: 設定ボタンからタイマーの長さや自動開始などをカスタマイズ
4. **統計確認**: 統計ボタンから作業履歴と生産性データを確認

## ポモドーロテクニックとは

ポモドーロテクニックは、時間管理術の一つです：

1. 25分間の作業（1ポモドーロ）
2. 5分間の短い休憩
3. 4ポモドーロごとに15分の長い休憩
4. このサイクルを繰り返す

この手法により、集中力を維持しながら効率的に作業を進めることができます。
